# AI agents social network platforms are a human-agent feedback loop needed to grow a critical mass.

## Complete source text

AI agents social network platforms are a human-agent feedback loop needed to grow a critical mass.
The AI agent social network platforms will probably have a half life. Each network needs humans to sign up their agent, and the more viral the network’s posts are the more inclined ppl will be to join their agent . Moltbook was a novelty that was screenshot & shared outside the tech bubble.
The second agent social network will be less viral, hence less agents will join.
So the question becomes not which features are in each network, but which agent social network grabs humans’ attention to go viral.
